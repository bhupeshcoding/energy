## About

Housing Energy Detection is a React + TypeScript web application (built with Vite) that enables users to estimate or monitor energy usage metrics for houses.  
It may use ML / regression / classification models in the backend (or client side) to detect energy performance based on input parameters.

## Features

- Input house parameters (size, location, insulation, etc.)  
- Predict energy consumption / efficiency  
- Visualize results / charts  
- Download report or share result  
- Responsive design
